# posts-ui
This application is built using [Vue.js](https://vuejs.org/).


Libraries Used:
* lodash
* Vuetify (UI styling)

## Project setup
```
npm install -g @vue/cli
```
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```
### Runs Unit Tests

```
npm run test:unit
```


### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
